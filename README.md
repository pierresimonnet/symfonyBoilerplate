# Symfony Boilerplate
Start a Symfony project easily

## In this boilerplate
- Symfony 5
- API Plateform
- Webpack Encore
- Liip Imagine
- PHPStan
- CSS Reset file

## Install : 
`composer install`
`yarn install`

## Help : 
`make help`

## Liip Imagine
Add / remove filters : config/packages/liip_imagine.yaml
